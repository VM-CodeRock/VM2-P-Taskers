About NUWCDIVNPT & Code 107

The Naval Undersea Warfare Center Division Newport (NUWCDIVNPT) is the Navy's full-spectrum research, development, test and evaluation (RDT&E) organization for undersea warfare, headquartered at 1176 Howell Street, Newport, Rhode Island. NUWCDIVNPT serves as the Navy's lead laboratory for submarines, autonomous underwater systems, and offensive/defensive undersea weapon systems, with a workforce of approximately 4,000 civilian and military personnel. (LinkedIn — NUWC Newport)

Code 107 — Security Division is part of the Code 10 Corporate Operations Department at NUWCDIVNPT. Code 10 provides timely and cost-efficient business, tactical, and strategic services to the command. Based on the organizational chart, Code 107 falls under Code 105 (Security Division) area, managing physical security, special security operations, and compliance. Historical contract data shows Code 107 has been supported with professional and administrative services covering MILCON (military construction), safety and occupational health, explosives safety, facilities management, and property management — all compliance-heavy functions requiring both technical and administrative professional support. (NAVSEA — NUWCDIVNPT Codes & Departments)

PAS (Professional and Administrative Services) in this context encompasses the full spectrum of support services needed to sustain Code 107's corporate operations functions: program management support, financial management, technical writing, planning support, administrative work center staffing, document processing, data maintenance, and facilities/property management compliance support. The estimated team size is 3–5 FTEs with SCA wage determinations applicable. (Drexault Contract Intelligence)
