Requirement Description
Official Scope: Provides the TSA with Operational Test and Evaluation activities, agency-wide field demonstration, data collection and analysis services. Includes developmental activities that enable test preparation, test execution, development of test articles, verification of test articles, data capture, test analysis, and related engineering change reviews. Provides test personnel to support test execution in airports, subject matter experts for cyber resilience testing and human factor experts for human system integration assessment. Provides essential technical resources to conduct operational testing at airport locations for system evaluation of new and improved security screening technologies, including credential authentication systems, explosives detection, prohibited items detection and alarm resolution in checkpoint and checked baggage screening.

This is TSA's primary enterprise-wide OT&E contract vehicle for the Test and Evaluation Division (TED), covering the full lifecycle from developmental test planning through independent operational assessments. The scope encompasses:

  • OT&E Lifecycle: Developmental activities, test preparation, test execution, test article development and verification, data capture, and analysis
  • Field Demonstrations: Agency-wide field demos, airport-based operational testing at CONUS and OCONUS locations
  • Technology Domains: Credential authentication systems (CAS), explosives detection systems (EDS), prohibited items detection, alarm resolution — checkpoint and checked baggage
  • Specialized Support: Cyber resilience testing SMEs, human systems integration (HSI) assessments, engineering change review participation
  • Cross-Agency: Coordination with DHS partners (CWMD, S&T) for co-sponsored testing programs
