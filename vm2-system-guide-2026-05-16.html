Disconnected Connectors — REST-First Rule (v2.6 — NEW 5/15)

When a connector is disconnected, all VM2 operations must fall back to direct REST API calls. No cron or audit may treat a disconnected connector as a compliance failure or escalation trigger.

SAM Monitor Hardening (v2.6 — Applied 5/15)

The SAM.gov daily monitor script (/home/user/workspace/data/sam/daily_monitor.py) was patched on 5/15 to prevent concurrent-run SQLite lock errors:

1
WAL mode + busy timeout: PRAGMA journal_mode=WAL; and PRAGMA busy_timeout=30000; (30 seconds)
2
Connection timeout: sqlite3.connect(db_path, timeout=30.0)
3
Lockfile with stale-PID detection: /home/user/workspace/data/sam/.daily_monitor.lock — checks if PID from lock is still active before honoring it
Trigger: Concurrent-run SQLite-lock error escalated 5/15 at 9:18 AM. If SQLite shows a run with status='in_progress' but no process owns it (lockfile absent or PID dead), run: UPDATE runs SET status='failed' WHERE status='in_progress'; before the next scheduled run.

Gmail Attachment Retrieval (v2.5 — Confirmed Pattern)

When a task email contains attachments (PDFs, spreadsheets, images), VM2 uses a three-step exhaust-order:

1
get_email_attachments(email_id) — Returns attachment list with IDs. Always attempted first.
2
read_email(email_id, include_attachments=true) — If step 1 returns empty.
3
download_attachment(attachment_id) — Downloads specific attachment once ID is obtained.
Confirmed 5/13: Used for BAFM PDF/xlsx and WashExec PNG. This is the canonical pattern for all Gmail attachment retrieval.

Thread Tag System (v2.3)

Threads group related deliverables into named topic clusters. A thread tag is a lightweight slug (e.g., faa-apex) that binds deliverables together across sessions.

1
Gmail email task: include [thread:slug] anywhere in subject line. Intake cron auto-extracts and injects into deliverable and portal row.
2
Todoist task: apply vm2-thread label and put thread:slug as first line of task description.
3
View thread: open the aggregator page, e.g., faa-apex-thread.html. Always up to date.
Active threads (5 with aggregator pages): faa-apex, faa-atlas, faa-oig-47238, mkt-aic, oasis-plus. 9 total thread tags in use.
