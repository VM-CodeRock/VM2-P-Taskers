How to share

  1. Copy any URL above and send it to whoever needs access.
  2. The recipient clicks the link → goes straight to the portal, no login.
  3. Once they've clicked it, a cookie is set for 7 days. They can navigate to any subpage without keeping ?key= in the URL.
  4. If you ever want to cut access, remove that key from VM2_PUBLIC_KEYS on Railway → service redeploys → old links 401.

Rotate a key

  1. Generate a strong key in your terminal: openssl rand -base64 24 | tr -d '/+=' | cut -c1-24
  2. Or use the button below to generate one client-side.
  3. Open Railway → vm2-portal service → Variables tab.
  4. Edit VM2_PUBLIC_KEYS — comma-separated list. Keep the previous key for a few days as a grace window, then drop it on the next rotation.
  5. Railway redeploys in ~30s. New URL is live.

⚠ This generates a key client-side for convenience. Add it to Railway env to activate.
