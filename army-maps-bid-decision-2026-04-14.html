Assessment of the Proposal Manager's Approach

Overall Assessment: Workable but High-Risk — 3 Concerns Need Immediate Attention

A 17-day window for a $50B IDIQ submission requiring government-signed QP certifications, CPARS pulls, and HR data analytics is aggressive by any standard. Teha's involvement as Senior Proposal Writer is the right move. However, three issues create structural risk regardless of how well the schedule is constructed:

Concern 1 — CO Signature Lead Time (Critical)

Every QP (up to 3 per domain) requires a dual signature from the offeror AND a Contracting Officer or designated customer representative per Attachment 0003. Missing government signatures automatically disqualify the QP. With 17 days total, and assuming 5–7 business days to get CO signatures from ICE, FAA, and CBP contracting teams, there is no margin for error. Teha needs to have QP packages sent to COs today or tomorrow (April 14–15) to ensure signed returns by April 22–23.

Concern 2 — Parallel Submissions for Multiple Domains + Multiple Entities

If bidding both Changeis ELB and ArchZen SB, and targeting D3 + D4 for each (potentially D5 for ArchZen), that is up to 4–6 separate proposals. The Volume II Scorecard must be unique per domain per entity, though Volume III (QPs) and Volume I (cover letter) can be reused. Teha's schedule must account for this parallel track explicitly. If it doesn't, the team will realize the multiplication too late.

Concern 3 — Domain Split Decision Must Be Made Now

Teha's note reminds that Changeis and ArchZen cannot submit in the same domain. This is not a minor housekeeping item — it determines which QPs go where, which certifications/systems are claimed by which entity, and how vacancy rate data is attributed. The domain split must be locked before any proposal writing begins. See Section 3 for the full analysis.

Section 3 · Domain Split Decision — Changeis vs. ArchZen NEW · CRITICAL

Teha's email flagged the key constraint: "Changeis and ArchZen are not permitted to submit in the same domain." This is standard for MPJV arrangements and aligns with SBA 13 CFR 125.2 joint venture affiliation rules — if both entities propose in the same domain, the Army will count Changeis's revenue against ArchZen's SB size standard, potentially disqualifying ArchZen. This makes the domain split the most strategically important decision in this entire bid.

Non-Negotiable Rule: Each Domain → One Entity Only

If Changeis submits in Domain 3 (ELB track), ArchZen cannot submit in Domain 3 (SB track). And vice versa. With up to 5 domains, the question is which 2–3 domains go to Changeis (ELB) and which 2–3 go to ArchZen (SB). The answer should be driven by which entity has the stronger QP portfolio for each domain, and which set-aside tier has better odds per domain given anticipated competition.
