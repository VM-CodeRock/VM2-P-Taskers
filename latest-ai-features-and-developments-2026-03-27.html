1) Google: Gemini inside Workspace (Docs/Sheets/Slides/Drive)

What changed

  • Docs: Generate first drafts from a prompt that can pull from relevant files; new tools to match writing style and match doc format (e.g., fill an itinerary template using details from emails).
  • Sheets: Create or extend sheets from prompts, including using email/file context; “Fill with Gemini” can generate text/categorize/summarize and can also pull in real-time web info (e.g., populate trackers with due dates/tuition).
  • Slides: Generate editable slides aligned to the deck theme and context; whole-deck generation is described as coming soon.
  • Drive: AI overview at top of Drive search results; “Ask Gemini in Drive” can answer questions across selected files (with citations).

Availability

  • Rolling out in beta starting Mar 10, 2026; initially for Google AI Ultra and Google AI Pro subscribers.
  • Docs/Sheets/Slides: English globally; Drive: U.S. (per Google’s note).

GovCon founder angle: This is very close to “AI-native proposal operations” inside the tools teams already live in. Use it for L0 standardization (tone, structure, template compliance) before human review.

Do this week (high ROI)

  1. Proposal template enforcement: Use Gemini in Docs to match style/format across boilerplate sections; treat it as an L0 editor before compliance review.
  2. Capture/pipeline tracker: Prompt Gemini in Sheets to build a pursuit tracker + contact list grounded in your inbox; then lock that sheet as the standard.
  3. AI adoption learning curve: Run a 30-day internal habit program (15 min/day) for recurring work (capture notes → call plan → draft email → risk register → exec summary). Track speed + quality.

2) Google: Gemini app — import “memories” + chat history

Gemini added switching tools (Settings) to import personal context, preferences, and chat history from other AI apps via a prompt or ZIP file upload; “past chats” is renamed to “memories.”

Why it matters: Lower switching costs. You can consolidate your working context into one assistant instead of re-briefing every time.
